## Overview

This repository contains practical implementations and exercises covering fundamental and advanced Java programming concepts. Each section builds upon previous knowledge, demonstrating progressive learning through hands-on coding.

## Content Structure

### Fundamentals (Sections 4-7)
- Input/Output operations and data handling
- Control structures and conditional logic
- Functions and methods implementation
- String manipulation and Math utilities

### Object-Oriented Programming Basics (Sections 8-9)
- Class design and object instantiation
- Constructors and initialization
- Encapsulation and access modifiers
- Static members and utilities

### Collections & Data Structures (Section 10)
- List and ArrayList implementations
- Boxing/Unboxing and Wrapper classes
- forEach loops and iterators
- Practical exercises with collections

### Advanced Topics (Sections 11-13)
- Comparators and sorting algorithms
- Enumerations and type safety
- Inheritance and class hierarchies
- Polymorphism and dynamic binding
- Abstract classes and methods
- Method overriding and super keyword
- Upcasting and downcasting
